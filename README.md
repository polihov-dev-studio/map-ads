# Карта лифтов с рекламой

Простой статический сайт для GitHub Pages.

## Что умеет

- карта Оренбурга на OpenStreetMap;
- добавление адреса через поиск;
- добавление точки кликом по карте;
- статусы:
  - `Не размещена`;
  - `Размещена`;
- сохранение всех точек и статусов в `localStorage`;
- сброс всех статусов без удаления адресов;
- удаление отдельной точки;
- экспорт/импорт JSON для переноса данных на другой телефон или компьютер.

## Как запустить на GitHub Pages

1. Создай новый репозиторий на GitHub.
2. Загрузи в него файлы:
   - `index.html`
   - `style.css`
   - `app.js`
   - `points.js`
3. В настройках репозитория открой:
   `Settings` → `Pages`
4. В `Build and deployment` выбери:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Нажми `Save`.

Через минуту сайт будет доступен по ссылке GitHub Pages.

## Как заранее добавить адреса вручную в код

Открой файл `points.js` и добавь точки в массив `DEFAULT_POINTS`.

Пример:

```js
const DEFAULT_POINTS = [
  {
    id: "tereshkovoy-10",
    address: "Оренбург, ул. Терешковой, 10",
    lat: 51.7875,
    lng: 55.1012
  },
  {
    id: "pobedy-75",
    address: "Оренбург, пр. Победы, 75",
    lat: 51.7721,
    lng: 55.1245
  }
];
```

Если ты уже пользовался сайтом и потом добавил новые точки в `points.js`, они тоже появятся. Старые сохранённые статусы не потеряются.

## Важно

Данные сохраняются только в браузере, где ты работаешь. Если открыть сайт на другом телефоне или компьютере, там будет пусто, пока не сделать импорт JSON.
